    【云服务器】云服务器使用

    网站作者：杜小白 2022/9/11 Linux云服务器

    @TOC

    不同的服务器和不同的系统版本可能会导致操作有些许不同，如果你是阿里云或者腾讯云又或者是百度云的用户，请自行探索，大致的思路是一样的。

    我个人相关配置： 某平台云服务器 CentOS 7.6 x86_64(64bit)(宝塔面板) MacOS

    # 1.云服务器

    对于大多数新手来说会感到很陌生，个人觉得服务器就相当于一台电 脑，购买完成你会有此服务器（电脑）的控制权、使用权。注：购买服务器有各种配置（CPU、内存大小、操作系统等）的选择，比如我购买的是服务器，1核1G内存的，这个可以根据个人需求进行选择购买。

    # 2.操作系统

    操作系统在我们购买云服务器的时候就需要根据需要进行选择，不用单独安装。但是如果在后期使用过程中，发现不合适，我们可以进行重装系统。

    # 3.基础配置

    此处我使用的是宝塔面板来进行管理 宝塔面板 (opens new window)可以进行服务器运维管理。

    （1）在控制台，重置密码等设置 （2）查看并打开几个常用端口

    （3）重启服务器

    # 3.连接服务器

    有服务器了我们就需要连接到购买的远程服务器（毕竟不是买一台电脑直接开机操作就可以，只是购买了网上能够自行操作的类似电脑的一个空间），连接方法有多种，下面说一下我使用的几种（可配合使用）：

    # 使用Terminal：

    （1）查看公网IP 在购买完成云服务器后，获得了公网IP。可以在控制台进行配置，默认端口是80. （2）确保云服务器的22端口打开，即SSH服务可以使用 （3）在terminal中输入： ssh 用户名@公网IP地址 例如：ssh root@123.124.124.125 然后输入密码

    # 其他方法：（mac为主）

    • [x] FinalShell (opens new window):免费、跨平台
    • [x] Electerm (opens new window)：类似于XShell。免费集终端模拟器、文件管理器、ssh远程连接、sftp客户端为一体的工具
    • [x] Termius (opens new window)：好用而且漂亮的SSH客户端，能快速远程控制服务器，可以定制自己喜欢的主题，支持FTP/SFTP连接。
    • [x] Royal TSX for Mac (opens new window):功能非常强大的远程连接管理工具。兼容多种连接类型，比如：RDP、VNC、基于SSH连接的终端，SFTP/FTP/SCP或基于Web的连接管理 ....等很多实用工具
