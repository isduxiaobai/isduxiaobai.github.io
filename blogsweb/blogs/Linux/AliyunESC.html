    # 阿里云ECS服务器搭建

    # 一、服务器搭建的网址入口

    # 有试用资格：

    点击进入阿里云云产品试用中心 (opens new window)，选择下图产品，选择试用30天。

    # 无试用资格：

    点击进入ecs主页 (opens new window)，如下图，点击立即购买，花费约为0.8.

    # 二、开始搭建

    # 1、选择基础配置 购买时，

    如图选择按量付费，选择cpu和内存：

    • 如下图选择任意镜像+高效云盘40g，然后点击下一步

    # 2、选择网络和安全组

    选择按使用流量 设置为80M ，点击确认订单。

    # 3、跳转到管理控制台页面

    页面地址：点击进入 (opens new window)，选择试用或购买的服务器地理位置，呈现效果如下图。

    # 4、设置安全组

    如图操作，进入安全组页面 通过百度获取自己ip

    将8888端口号入口放开为你的ip（百度得到到ip）： 将80端口号放开入口：

    # 5、重置操作系统和镜像

    回到控制台页面，页面地址：点击进入 (opens new window)。然后如图关闭服务器 选择重置系统和镜像 如图点击从镜像市场选择操作系统

    选择宝塔 设置自定义密码并确认订单

    # 6、链接服务器，获取宝塔初始信息

    # 7、设置服务器环境

    进入宝塔控制面板后 , 进入软件商店,搜索tomcat后点击安装，选择tomcat9版本安装后, 如图搜索tomcat, 并设置端口号

    # 8、部署项目

    如图点击进入tomcat安装的文件夹 , 并进入webapps文件夹 将webapps中的ROOT文件夹内容清空，将我们的代码上传到ROOT
